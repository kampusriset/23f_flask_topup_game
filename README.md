# Nama Aplikasi 
Web Top UP Game (masih dasar)

# Kegunaan Web 
Website top up game ini berfungsi sebagai platform layanan pembelian saldo atau item game secara online yang memudahkan pengguna dalam melakukan transaksi top up dengan cepat, aman, dan terorganisir. Website ini menghubungkan pengguna (user) dan pengelola (admin) dalam satu sistem terintegrasi berbasis web.
